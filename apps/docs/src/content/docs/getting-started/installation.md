## Installation

### Install tokens

```bash
npm install @spinozaos/tokens
# or
pnpm add @spinozaos/tokens
```

### Install React components

```bash
npm install @spinozaos/react
# or
pnpm add @spinozaos/react
```

### Install Motion presets

```bash
npm install @spinozaos/motion
# or
pnpm add @spinozaos/motion
```

### Install Tailwind preset

```bash
npm install @spinozaos/tailwind
# or
pnpm add @spinozaos/tailwind
```

## Peer Dependencies

The React package requires:

- `react` >= 18.0.0
- `react-dom` >= 18.0.0
- `framer-motion` >= 10.0.0
